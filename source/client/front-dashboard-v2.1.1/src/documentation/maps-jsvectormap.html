Maps (JSVectorMap)

A lightweight JavaScript library for creating interactive maps and pretty data visualization.

JSVectorMap documentation

How to use

Copy-paste the stylesheet <link> into your <head> to load the CSS.

        
          <link rel="stylesheet" href="./node_modules/jsvectormap/dist/css/jsvectormap.min.css">

Copy-paste the following <script> near the end of your pages under JS Implementing Plugins to enable it.

        
          <script src="@@autopath/node_modules/jsvectormap/dist/js/jsvectormap.min.js"></script>
          <script src="@@autopath/node_modules/jsvectormap/dist/maps/world.js"></script>

Copy-paste the following <script> near the end of your pages under JS Front to enable it.

        
          <script src="./assets/js/hs.jsvector-map.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

        
          <script>
            (function() {
              // INITIALIZATION OF VECTOR MAP
              // =======================================================
              HSCore.components.HSJsVectorMap.init('.js-jsvectormap')
            })();
          </script>
